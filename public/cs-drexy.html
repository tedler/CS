drexy

Buy-Sell-Swap mobile application for iOS & Android platforms

branding UX design UI design front-end

brief

The main goal was to create buy-sell-swap marketplace for fashion outfits and secondhand clothing. Design a new way to sell wardrobe out, buy new clothes and look good without spending a fortune. Make awesome, stylish and everyday use apps for iOS and Android platforms.

branding

Starting with the logo, my challenge was creating clean and unique style to highlight modern tunes of online fashion industry. Sets of required UI and category icons were designed to bring a more "human" feel to the product.

user flow mapping

After the user research and exploration of main usage scenarios, I switched to rapidly wireframing and testing the user flows in order to define key functionality, information architecture and interaction patterns.

detailed wireframes

Based on the final version of user-flow mapping I sketched out and created detailed wireframes for the key pages of the product before going to building interactive prototype and user testing stage. Some of wireframes can be seen below.

ui style guide

In parallel with designing wireframes and the brand identity, I also established the styles for all required UI elements and created detailed UI style guide to offer a powerful set of tools for development team.

layout designs

Around 60 high-fidelity layout designs and all required assets were created for iOS and Android. Each page and each UI element was designed with love to build slick, powerful and interactive application.
