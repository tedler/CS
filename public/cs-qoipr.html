QOIPR

IT resellers web-based platform for desktop & tablet

branding UI design variations front-end

brief

My role was pure creative to polish existing branding up, give the entire project a makeover and create UI style guide to move forward with a consistent look and feel for QOIPR - web-based platform which helps IT resellers to handle their day-day business activities.

branding

I designed the brand identity to perfectly fit the project guideline: improved the logo prominence, polished the color scheme, got the better version of the type-based icons, and the time-proven Roboto was used as the main font.

ui style guide

Moving forward for the QOIPR project meant it needed a consistent and well structured UI style guide. The general idea was to keep UI elements really simple and call to actions bright and solid, so we can keep all new interfaces clean and to the point.

layout variations

10 different layout variations were created to improve user experience and get the balanced look and feel. The visual designs were responsive and contained several options of navigations, header structures and color schemes.
