Booking Direct

Booking engine for hotels with social media marketing platform

branding UX design UI design front-end

brief

My job for this project was to re-design the process of room booking, improve user experience, establish a style guide and provide new html/css prototype for BookingDirect - booking engine for hotels with dynamic pricing and social media marketing platform.

branding

I started on the branding to create all future components in one style. Icons set, color scheme and crisp typography using Gill Sans font were designed to give its own product individuality.

wireframing

In parallel with branding to save time I quickly sketched out some ideas on how required pages would function. Then proper wireframes for accommodation description page, 3 steps of booking and confirmation page were created as lean as possible to make the experience simple and frictionless.

ui style guide

Creating of user interfaces for this project was pretty simple, as most of the hard work had been done to design and finalise the branding and wireframes earlier.

design layouts

The main focus when creating design concepts was to make sure that the page layouts were clean and functional as possible, input forms were not scary to fill out and any unnecessary distractions were removed.

front-end development

I really wanted to make it easy for users to go through the accommodation booking process and my main goal was designing and building clean and fast set of UI components for development team.
